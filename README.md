# Ім’я: Нікіта

Ця гра про захист Землі від вторгнення прибульців. Гравець керує космічним винищувачем, який може рухатися горизонтально внизу екрану та стріляти лазерними пострілами вгору. З космосу наступають строї ворожих кораблів, які рухаються вліво-вправо та поступово спускаються донизу, час від часу відкриваючи вогонь по гравцю.

## Як запустити гру локально

- Відкрити файл `space-invaders.html` подвійним кліком у будь-якому сучасному браузері.
- Або запустити простий локальний сервер:
  ```sh
  python -m http.server 8000
  ```
  Потім перейти за адресою `http://localhost:8000/space-invaders.html`.
